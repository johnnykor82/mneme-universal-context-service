  priorities. Passing alpha tests do not imply v0 compliance when the tested
  behavior is narrower than the final spec.
- Status values: `COMPLIANT`, `PARTIAL`, `MISSING`, `UNCLEAR`,
  `OUT_OF_SCOPE/FUTURE`.
- Priority values: `BLOCKER`, `HIGH`, `MEDIUM`, `LOW`. For compliant rows,
  `LOW` means no current implementation gap was found in this audit slice.
- This audit did not inspect live Hermes, live `hermes-mneme`, or Mneme SQLite
  as an external contract substitute.
