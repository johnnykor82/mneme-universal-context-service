Mneme Universal Context Service

Local-first event memory for long-running tool-using agents. Raw events are preserved, indexed, and assembled into bounded request-only context when an agent needs continuity beyond the model window.

Core Promise

  • Raw agent events remain preserved.
  • Context is assembled per request, not persisted over transcript.
  • Agents can search, fetch, and expand memory evidence.

Cost Controls

  • Minimal mode works without embeddings or LLM enrichment.
  • Standard mode adds local embeddings and hybrid retrieval.
  • Quality mode adds rerank, graph expansion, and enrichment.

Adoption Path

  • REST API for generic runtimes.
  • MCP server for Codex, Claude Code, and compatible clients.
  • Reference adapters for Hermes, LangGraph, and OpenAI Agents SDK.
